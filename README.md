# Mustapha Ishmael - Portfolio Website

A modern, responsive portfolio website built with PHP, HTML, CSS, and JavaScript. This is the converted version of the original Next.js portfolio, maintaining all the functionality and design while using traditional web technologies.

## 🚀 Features

- *Responsive Design*: Works perfectly on desktop, tablet, and mobile devices
- *Modern UI*: Dark theme with yellow accent colors
- *Interactive Navigation*: Smooth section transitions with active state indicators
- *Contact Form*: Functional PHP backend with email sending and validation
- *Progress Bar*: Visual reading progress indicator
- *Typing Effects*: Animated command-line style text
- *Social Links*: Integrated social media profiles
- *Portfolio Sections*: About, Resume, Portfolio, and Contact sections
- *GitHub Integration*: Ready for GitHub stats integration
- *SEO Optimized*: Meta tags, Open Graph, and Twitter Card support
